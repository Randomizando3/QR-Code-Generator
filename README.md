# QR-Code-Generator

## English
AutoHotkey script that creates a QR code image from user-provided text or URLs.

### Features
- Simple desktop interface
- QR code generation from text input
- Local output as `qr-code.png`

### Usage
1. Install AutoHotkey.
2. Run `QR CODE Generator.ahk`.
3. Enter the text or URL you want to encode.
4. Click `Submit` to generate the image.

## Português
Script em AutoHotkey que cria uma imagem de QR Code a partir de um texto ou URL informado pelo usuário.

### Funcionalidades
- Interface simples de desktop
- Geração de QR Code a partir de texto
- Saída local como `qr-code.png`

### Uso
1. Instale o AutoHotkey.

3. Informe o texto ou a URL que deseja codificar.
