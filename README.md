# QR-Code-Generator

## English
QR-Code-Generator is an AutoHotkey script that creates a QR code image from user-provided text or URLs.

### Features
- Simple desktop interface
- QR code generation from text input
- Local output as `qr-code.png`

### Usage
1. Install AutoHotkey.
2. Run `QR CODE Generator.ahk`.
3. Enter the text or URL you want to encode.
4. Click `Submit` to generate the QR code image.
