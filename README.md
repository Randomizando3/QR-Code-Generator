# QR Code Generator

Script em AutoHotkey para gerar um QR Code a partir de um texto ou URL informados pelo usuario.

## Funcionalidades

- Interface simples para colar o texto
- Geracao automatica da imagem do QR Code
- Exportacao local em `qr-code.png`

## Como usar

1. Instale AutoHotkey.
2. Execute o arquivo `QR CODE Generator.ahk`.
3. Informe o texto ou URL desejado.
4. Clique em `Submit` para gerar o QR Code.

## Observacao

O script utiliza um servico externo para montar o QR Code, entao a geracao depende de acesso a internet.
